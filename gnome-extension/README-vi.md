# GNOME extension (Phase 4 — chưa bắt đầu)

> 🌐 English: [README.md](README.md)

Extension GNOME Shell (GJS) hiển thị chi phí "native" trên top-bar/panel của Ubuntu.
Đây là tương đương gần nhất với widget hệ thống trên GNOME (Linux không có WidgetKit).

Extension đọc snapshot JSON do lõi Rust ghi sẵn (xem [`../shared/SCHEMA-vi.md`](../shared/SCHEMA-vi.md))
— KHÔNG tự chạy ccusage. Chỉ chạy trên GNOME.
